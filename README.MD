# Spring Boot 학습용 프로젝트

### 1. 프로젝트 소개
본 프로젝트는 "Spring Boot Up & Running , 마크 헤클러 지음" 교재를 기반으로 개발한 웹 애플리케이션 입니다.<br>
프로젝트의 목표는 Spring Boot의 실무적인 이해와 클라우드 네이티브 애플리케이션 구축 학습에 있습니다.


### 2. 기술 스택
-   **Backend:** Java

-   **Framework:** Spring Boot 3, Spring Data JPA

-   **Database:** H2

-   **Build Tool:** Gradle

-   **View:** Thymeleaf
